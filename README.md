# Web Development Projects

This repository contains various web development exercises and solutions. Each folder represents a different topic or lesson, with example HTML and CSS files.

## Structure

- **2.1 Heading Element/**: Exercises and solutions for using HTML heading elements (h1-h6).
- **5.1. Adding CSS/**: Examples of adding CSS to HTML using inline, internal, and external stylesheets.

## How to Use

- Browse each folder to find the relevant exercise or solution.
- Open the HTML files in your browser or code editor to view and learn from the examples.

## Contributing

More exercises and solutions will be added over time. Feel free to fork this repository and practice your own web development skills!
